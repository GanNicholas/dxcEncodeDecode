## Getting Started

### To run the code
1) Change directory to the root directory of repo.
2) Build the java files
   <br/>
    ```
    javac .\EncodeDecodeMain.java .\EncodeDecodeRunner.java 
    ```
   <br/>

3) Run the class files
   <br/>
    ```
    java EncodeDecodeMain
    ```
   <br/>
